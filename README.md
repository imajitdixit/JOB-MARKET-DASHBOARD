## Overview
This dashboard provides insights into:

-  Python (Pandas, NumPy, Matplotlib)
-  Power BI (Interactive dashboard design)
-  Real-world job postings dataset

---

## Data Preprocessing
Performed in Jupyter Notebook:
- Cleaned and merged multiple CSVs  
- Removed duplicates, nulls  
- Standardized experience & salary fields  
- Extracted skills using a keyword dictionary  
- Exported final datasets for Power BI visualization
